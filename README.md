## README

Ferramentas de IA utilizadas: Lovable como acelerador da estrutura inicial e apoio visual, além do ChatGPT para refinamento de textos, responsividade e revisão de código.
A IA ajudou principalmente na conversão rápida da interface em componentes, organização da hierarquia visual e ganho de velocidade no processo inicial.
Na Parte 1, a IA acelerou a reprodução fiel do layout do Figma, reduzindo tempo na construção base da página.
Na Parte 2, a IA ajudou a explorar composições visuais para transformar o briefing em uma hero section mais alinhada à proposta.
Os ajustes manuais foram feitos no VS Code para corrigir espaçamentos, tipografia, responsividade e microdetalhes visuais.
Também refinei manualmente a semântica do HTML e pequenos comportamentos do CSS para melhorar consistência entre desktop e mobile.
Esses ajustes foram necessários para garantir maior fidelidade visual e um resultado final mais próximo de produção real.
